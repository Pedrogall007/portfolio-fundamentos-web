## 📌 Sobre o Projeto
Este é um projeto de Portfólio Pessoal Web desenvolvido como requisito prático para a disciplina de **Fundamentos da Programação Web** do curso de **Engenharia de Software**. 

O objetivo principal deste projeto foi construir uma aplicação "Single Page" (Página Única) totalmente responsiva e funcional, utilizando **exclusivamente tecnologias nativas** (Vanilla), sem o uso de frameworks externos como Bootstrap, Tailwind ou React.

🔗 **[Clique aqui para acessar o site em produção (Deploy via GitHub Pages)](https://pedrogall007.github.io/portfolio-fundamentos-web/)**

---

## 🚀 Funcionalidades Implementadas

O site atende a rigorosos critérios de avaliação técnica e usabilidade, incluindo:

- **Navegação por Âncoras (Smooth Scroll):** Menu fixo no topo que direciona o usuário para as seções correspondentes sem recarregar a página.
- **Alternador de Tema (Dark/Light Mode):** Funcionalidade nativa em JavaScript que manipula o DOM e variáveis CSS para alternar entre temas claro e escuro.
- **Validação de Formulário com Regex:** O formulário de contato possui validação estrita via JavaScript puro, impedindo envios vazios e verificando a formatação correta do e-mail.
- **Design Responsivo (Flexbox):** O layout (incluindo a grade de projetos do portfólio) se adapta perfeitamente a diferentes tamanhos de tela (Mobile, Tablet e Desktop).
- **Semântica HTML5:** Utilização de tags estruturais adequadas (`<header>`, `<nav>`, `<main>`, `<section>`).

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica de todo o conteúdo.
- **CSS3:** Estilização modular com variáveis (`:root`), Flexbox e Media Queries para responsividade.
- **JavaScript (ES6):** Lógica de validação de formulários, controle de eventos de clique e manipulação de classes CSS para o tema escuro.
- **Git e GitHub:** Controle de versionamento de código e hospedagem contínua via GitHub Pages.

## 📂 Estrutura das Seções

1. **Sobre mim:** Apresentação profissional, contexto de desenvolvimento e interesses técnicos.
2. **Formação:** Histórico acadêmico, cursos extracurriculares e proficiência em idiomas.
3. **Portfólio:** Repositórios e projetos práticos já desenvolvidos (com links externos).
4. **Contato:** Formulário interativo para envio de mensagens com validação e simulação de envio.
